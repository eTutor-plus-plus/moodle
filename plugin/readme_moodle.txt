# eTutor Sync API Plugin

After installing this plugin you have to perform following steps:

1. If not done so far, enable Moodle web services via Site Administration > Server > Web services.
2. Enable the REST protocol.
3. Create a web service user for the eTutor.
4. Add the webservice/rest:use capability to the created user.
5. Create a token for the user.
6. Enable the Multi-language content filter (Site Administration > Plugins > Filters > Manage Filters)
